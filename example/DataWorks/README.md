## 基本概念
维度是维度建模的基础和灵魂。在维度建模中，将度量称为“事实”
将环境描述为“维度”，维度是用于分析事实所需要的多样环境。例如，
在分析交易过程时，可以通过买家、卖家、商品和时间等维度描述交易
发生的环境。

度量/原子指标：支付金额。
维度：城市、时间。
